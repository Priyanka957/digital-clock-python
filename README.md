##Digital Clock using Python

A simple Digital Clock project built using Python.
This project displays the current time in real-time with a clean and easy-to-understand implementation.

Features
1.Real-time digital clock
2.Simple Python implementation
3.Beginner-friendly project
4.Easy to customize

##Technologies Used
1.Python
2.Tkinter (GUI Library)

##Project Structure
Digital-Clock/
│
├── clock.py
└── README.md

##How to Run
1.Clone the repository
    git clone https://github.com/Priyanka957/digital-clock-python.git
2.Open the project folder
    cd digital-clock-python
3.Run the Python file
    python clock.py

##Output

The application opens a digital clock window showing the current system time.
